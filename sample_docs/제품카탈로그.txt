테크스타 제품 카탈로그

1. FlowBot — 업무 자동화 플랫폼

FlowBot은 테크스타의 핵심 제품으로, 코딩 없이 드래그 앤 드롭 방식으로 업무 흐름을 자동화할 수 있는 플랫폼입니다.

주요 기능
- 워크플로우 빌더: 200개 이상의 앱과 연동 가능한 시각적 자동화 편집기
- 트리거 설정: 이메일 수신, 양식 제출, 시간 예약 등 다양한 이벤트 트리거
- 데이터 변환: 엑셀, CSV, JSON 간 자동 변환 및 정제
- 오류 감지 및 알림: 자동화 실패 시 Slack·이메일로 즉시 알림

요금제
- 스타터 플랜: 월 49,000원, 자동화 500건/월, 사용자 3명
- 비즈니스 플랜: 월 149,000원, 자동화 5,000건/월, 사용자 10명
- 엔터프라이즈 플랜: 별도 협의, 무제한 자동화, 무제한 사용자, 전담 CSM 배정

지원 연동 서비스: Gmail, Slack, Notion, 네이버웍스, 카카오워크, Salesforce, HubSpot,
Google Sheets, Microsoft 365, Jira, Zendesk 등 200여 가지

2. SmartFlow — AI 기반 지능형 자동화 플랫폼

SmartFlow는 2025년 3월 출시된 차세대 제품으로, 대형 언어 모델(LLM)을 결합하여
문서 이해, 데이터 추출, 자동 분류 등 AI 작업을 자동화 흐름에 통합할 수 있습니다.

주요 기능
- AI 문서 파서: PDF, 이미지, 스캔 문서에서 구조화된 데이터 자동 추출
- 자연어 조건 분기: "이 이메일이 불만 내용이면 → CS팀에 에스컬레이션" 같은 자연어 규칙 설정
- 자동 요약 및 분류: 대량 문서를 AI가 자동으로 요약·카테고리화
- 멀티 에이전트 오케스트레이션: 여러 AI 에이전트가 협력하여 복잡한 작업 수행

요금제
- 비즈니스 플랜: 월 299,000원, AI 처리 1,000건/월, 사용자 10명
- 엔터프라이즈 플랜: 별도 협의, 전용 LLM 파인튜닝 옵션 포함

3. DataSync — 실시간 데이터 통합 솔루션

DataSync는 사내 여러 시스템에 흩어진 데이터를 하나의 허브로 통합하고
실시간으로 동기화하는 솔루션입니다.

주요 기능
- CDC(Change Data Capture): 데이터베이스 변경사항 실시간 감지 및 전파
- 스키마 자동 매핑: 서로 다른 데이터 구조를 AI가 자동으로 매핑
- 데이터 품질 검증: 이상값, 중복, 누락 데이터 자동 탐지 및 알림
- 감사 로그: 모든 데이터 변경 이력 추적

지원 데이터베이스: MySQL, PostgreSQL, MongoDB, Oracle, MS SQL Server,
BigQuery, Snowflake, Amazon Redshift

요금제: 월 200,000원부터 시작, 데이터 볼륨 기준 과금

고객 사례

제조업 A사: FlowBot 도입 후 구매 발주 처리 시간 80% 단축
유통업 B사: DataSync로 ERP-CRM 연동, 재고 불일치 오류 95% 감소
금융업 C사: SmartFlow로 대출 신청서 자동 심사, 처리 속도 3배 향상
